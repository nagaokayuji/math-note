三次方程式の解の候補

三次方程式

ax3+bx2+cx+d=0ax^3 + bx^2 + cx + d = 0ax3+bx2+cx+d=0

を因数定理で解きたい

p,qp,qp,qを互いに素な整数として、

x=qpx = \frac{q}{p}x=pq​

とおくと、

a(qp)3+b(qp)2+c(qp)+d=0a(\frac{q}{p})^3 + b(\frac{q}{p})^2 + c(\frac{q}{p}) + d = 0a(pq​)3+b(pq​)2+c(pq​)+d=0
⇔aq3+bpq2+cp2q+dp3=0\Leftrightarrow aq^3 + bpq^2 + cp^2q + dp^3 = 0⇔aq3+bpq2+cp2q+dp3=0

変形すると

dp3=−p(bq2+cpq+dp2)(1)dp^3 = -p(bq^2 + cpq + dp^2) \tag{1}dp3=−p(bq2+cpq+dp2)(1)
aq3=−p(bq2+cpq+dp2)(2)aq^3 = -p(bq^2 + cpq + dp^2) \tag{2}aq3=−p(bq2+cpq+dp2)(2)

p,qp,qp,q は互いに素であるから

  • (1)(1)(1)より qqq は dddの約数
  • (2)(2)(2)より ppp は aaaの約数

となる。

よって三次方程式

ax3+bx2+cx+d=0ax^3 + bx^2 + cx + d = 0ax3+bx2+cx+d=0

の解の候補は

±dの約数aの約数\plusmn \frac{d\text{の約数}}{a\text{の約数}}±aの約数dの約数​
