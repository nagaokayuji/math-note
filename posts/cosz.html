解析学複素数
2021/07/16

cos⁡z=4\cos z = 4cosz=4

z は複素数

考えたこと

オイラーの公式から

eiθ=cos⁡θ+isin⁡θe^{i\theta} = \cos\theta + i\sin\thetaeiθ=cosθ+isinθ

変形すると

cos⁡θ=eiθ+e−iθ2\cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}cosθ=2eiθ+e−iθ​

より、与式は

eiz+e−iz2=4 \frac{e^{iz} + e^{-iz}}{2} = 42eiz+e−iz​=4
eiz+e−iz=8e^{iz} + e^{-iz} = 8eiz+e−iz=8

z=a+biz=a+biz=a+bi ( a,b∈Ra, b \in\mathbb{R}a,b∈R ) とすると

e2b⋅e−ai−8⋅eb+eai=0e^{2b}\cdot e^{-ai} - 8\cdot e^b + e^{ai} = 0 e2b⋅e−ai−8⋅eb+eai=0
eb=4±15e^b = 4 \pm \sqrt{15}eb=4±15​

よって b=log⁡(4±15)b=\log {(4\pm \sqrt{15})}b=log(4±15​) （純虚数の場合はこれでOK）

↑ このときの aaa の値を求めたい

解答

z=x+iy(x,y∈R)z = x + iy (x, y \in \mathbb{R}) z=x+iy(x,y∈R) とおくと、

cos⁡z=cos⁡(x+iy)=cos⁡xcos⁡(iy)−sin⁡xsin⁡(iy)\cos z = \cos(x+iy) = \cos x \cos (iy) - \sin x \sin (iy)cosz=cos(x+iy)=cosxcos(iy)−sinxsin(iy)
=cos⁡xcosh⁡y−isin⁡xsinh⁡y=4= \cos x \cosh y - i\sin x \sinh y = 4=cosxcoshy−isinxsinhy=4

より、

cos⁡xcosh⁡y=4\cos x\cosh y = 4cosxcoshy=4

また、

sin⁡xsinh⁡y=0\sin x \sinh y = 0sinxsinhy=0

より、 sin⁡x=0\sin x = 0sinx=0 または sinh⁡y=0\sinh y = 0sinhy=0 となる

求める解は

z=2nπ+ilog⁡e(4±15)z = 2n\pi + i\log_e{(4 \pm \sqrt{15})}z=2nπ+iloge​(4±15​)
