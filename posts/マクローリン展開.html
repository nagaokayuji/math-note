マクローリン展開

無限回微分可能な関数をxxxの整式で表したい。

f(x)=a0+a1x+a2x2+⋯+anxn+⋯f(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n + \cdotsf(x)=a0​+a1​x+a2​x2+⋯+an​xn+⋯

f(0)=a0f(0) = a_0f(0)=a0​

a0=f(0)a_0 = f(0)a0​=f(0)

1回微分すると、

f′(x)=a1+2a2x+3a3x2+⋯f'(x) = a_1 + 2a_2x + 3a_3x^2 + \cdotsf′(x)=a1​+2a2​x+3a3​x2+⋯

a1=f′(0)a_1 = f'(0)a1​=f′(0)

2回微分すると

f′′(x)=2a2+6a3x+⋯f''(x) = 2a_2 + 6a_3x + \cdotsf′′(x)=2a2​+6a3​x+⋯

nnn回微分すると、

となり、a\bm{a}aが求められる。

以上より、以下が成り立つ。

f(x)=f(0)+f′(0)1!x+f′′(0)2!x2+f′′′(0)3!x3+⋯+f(n)(0)n!+⋯=∑n=0∞f(n)(0)n!xnf(x) = f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + \cdots + \frac{f^{(n)}(0)}{n!} + \cdots = \sum_{n=0}^\infty \frac{f^{(n)}(0)}{n!}x^nf(x)=f(0)+1!f′(0)​x+2!f′′(0)​x2+3!f′′′(0)​x3+⋯+n!f(n)(0)​+⋯=n=0∑∞​n!f(n)(0)​xn
